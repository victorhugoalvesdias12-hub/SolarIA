# ☀️ SolarIA

Simulador inteligente de sistemas fotovoltaicos.

## 🚀 Funcionalidades

- Cálculo de potência do sistema
- Quantidade de placas
- Economia mensal e anual
- Payback (tempo de retorno)
- Simulação de telhado
- Otimização automática de layout
- Comparação custo x economia (gráfico)

## 🧠 Tecnologias

- React (Vite)
- Recharts

## ▶️ Como rodar

```bash
npm install
npm run dev
